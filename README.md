# PythonCodeFlowVizualizer

Visualizador de flujo de código python.

## Ejecución remota

```
curl -sL https://raw.githubusercontent.com/nipegun/PythonCodeFlowVisualizer/refs/heads/main/pcfv.py | python3 - -- [Script]
```

...donde [Script] es la ruta (absoluta o relativa) hacia un script de python.
